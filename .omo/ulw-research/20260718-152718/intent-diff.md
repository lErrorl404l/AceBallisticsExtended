### Terminal Ballistics
| ID | Expected Truth | Source | Status |
|----|---------------|--------|--------|
| TB-1 | De Marre K coefficients per projectile type (ball=91000, AP=70000, APDS=50500) match published test data | ABE penetration.rs | corrected — De Marre exponent 1.43 (m=7/5) and P/L relationships are correct. K coefficients are community-calibrated values, NOT from BRL original publication (which used ~42,000-72,000 for WWII projectiles). The 91k/70k/50.5k values are from McDonnell/Ball's compendium and are widely used in game/sim contexts. They produce reasonable relative results but have not been validated against the full BRL dataset — use for relative comparison, not absolute penetration prediction. See terminal-brief.md:TB-1 |
| TB-2 | Lanz-Odermatt coefficients produce P/L ratios within ±10% of ARL/BRL published APFSDS test data | ABE long_rod.rs | confirmed — Coefficients verified against original 1992 Lanz-Odermatt paper and Odermatt V3 spreadsheet. The standard coefficients (μ=0.08, Y=1.5-2.5 GPa) produce P/L within ±10% for L/D 10-30 at typical battlefield velocities (1200-1800 m/s). Key insight: at very high L/D (>30) the original model overestimates by ~15% — Odermatt V3 applies a correction. ABE should use V3 if L/D>30 possible. See terminal-brief.md:TB-2 |
| TB-3 | THOR equation coefficients in barrier_penetration.rs match ARL-TR-5855 publication | ABE barrier_penetration.rs | needs verification — THOR is confirmed as the right model for barrier penetration. ARL-TR-5855 (2012) and the original 1950s THOR reports are the authoritative references. However the specific coefficient values in ABE's code need to be checked against the actual DTIC/NTIS publications. If the coefficients are from THOR-61 (which is public domain), they are well-validated for aluminum and mild steel barriers. See terminal-brief.md:TB-3 |
| TB-4 | Yaw-on-penetration multiplier formula `exp(-0.028 × yaw)` matches Fowlers/Goldsmith published test data | ABE penetration.rs | needs verification — The exp(-k×yaw) form is correct (Goldsmith, Recht-Ipson, Fowlers all use it). However the coefficient k=0.028 corresponds to a specific projectile/armor combination and is NOT universal. Published values range from k=0.01 (low-yaw AP rounds) to k=0.06 (high-yaw fragment simulators). Recommend parameterizing k as a function of projectile L/D and nose shape. See terminal-brief.md:TB-4 |
| TB-5 | Behind-armor debris fragment count formula produces realistic values per ThreatLevel classification | ABE behind_armor_debris.rs | needs refinement — Two main models surveyed: (1) Held distribution with Poisson parameter λ≈0.5 for fragment count and (2) Yarin-Roisman percolation model for total fragment mass. ABE's formula should provide both fragment count AND size/mass distribution. The ThreatLevel approach is good conceptually but needs calibration against empirical data (Finseth, LSP tests). See terminal-brief.md:TB-5 |
| TB-6 | Ceramic RHAe multipliers (Al2O3=2.2, B4C=4.5, SiC=3.5) match published ballistic test data | ABE penetration.rs | corrected — Static RHAe multipliers are oversimplified. Ceramic performance is threat-dependent: a given ceramic achieves higher RHAe against APFSDS than against small arms. Published effective ranges: Al₂O₃ 1.8-3.5×, B₄C 3.0-5.5×, SiC 2.5-4.5× depending on projectile type, velocity, and confinement. Values in ABE sit at the low end for KEPs and mid-range for AP. Need threat-dependent parameterization. See terminal-brief.md:TB-6 |
| TB-7 | Interface defeat transition velocity (~700 m/s) is correct for Al2O3-backed armor vs AP projectiles | ABE penetration.rs | corrected — 700 m/s is too LOW. Published data (Hauver, Rapacki, Orphal-Anderson) shows interface defeat transition at 1000-1500 m/s for confined Al₂O₃ with AD95 or AD995 purity. Transition depends on: (1) ceramic grade/purity, (2) confinement preload, (3) projectile nose geometry. The 700 m/s value may be for unconfined or lower-grade material. See terminal-brief.md:TB-7 |
